# YouTube to Shorts Generator dengan Subtitle Otomatis

Aplikasi berbasis web sederhana untuk memotong video YouTube menjadi format vertikal (9:16) dan menambahkan subtitle bahasa Indonesia otomatis menggunakan AI Whisper.

## Cara Install & Menjalankan di Komputer Lokal

1. Pastikan Anda sudah menginstal **FFmpeg** dan **ImageMagick** di sistem operasi Anda.
2. Clone repositori ini atau download kodenya.
3. Instal semua library Python yang dibutuhkan:
   ```bash
   pip install -r requirements.txt
   ```
4. Jalankan aplikasi:
   ```bash
   streamlit run app.py
   ```
